# 📱 Pokédex App

A simple and interactive Pokédex application that allows users to search, view, and explore Pokémon with detailed information such as stats, abilities, and types.

---

## 🚀 Features

- 🔍 Search for any Pokémon by name
- 📊 View detailed stats (HP, Attack, Defense, etc.)
- ⚡ See Pokémon types and abilities
- 🖼️ Display official Pokémon sprites/images
- 📱 Responsive design (mobile & desktop)
- ⚡ Fast data fetching using API

```bash

```

### 2. Navigate into the project folder
```bash

```

- For HTML:
  Open `index.html` in your browser

- For React:
```bash
npm install
npm start
```
